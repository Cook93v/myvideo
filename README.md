# Quiz TikTok Maker

Site complet React + Vite prêt à héberger sur Netlify.

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Netlify

- Build command : `npm run build`
- Publish directory : `dist`

Le fichier `netlify.toml` est déjà présent.

## Fonctions

- Création de séquences intro / question / fin
- Modification de chaque séquence
- Background par séquence : thème, couleur, image perso
- Export / import JSON
- Sauvegarde auto navigateur
- Génération vidéo `.webm`
- Aperçu vertical style TikTok
